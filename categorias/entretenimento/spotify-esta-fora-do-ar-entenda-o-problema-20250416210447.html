Spotify está fora do ar, entenda o problema

entretenimento 16/04/2025 às 21:04
Imagem: BrasilViral

O serviço de streaming de música Spotify está enfrentando problemas técnicos em diversas partes do mundo. Os usuários da plataforma, tanto os assinantes quanto os que utilizam a versão gratuita, relataram dificuldades em acessar o serviço na manhã de quarta-feira. De acordo com o Spotify, o problema não está relacionado a um ataque cibernético.

Os primeiros relatos de interrupção foram feitos por usuários nos Estados Unidos e no Reino Unido, mas logo o problema se espalhou para outros países, incluindo o Brasil. O Spotify se manifestou oficialmente, informando que estava ciente dos problemas e que estava trabalhando para resolver a questão. No entanto, a empresa não forneceu detalhes sobre a causa do problema ou uma previsão para quando o serviço regular seria restaurado.

Os problemas estão afetando diferentes plataformas do Spotify, incluindo o player para navegadores, o aplicativo para desktop e o aplicativo para iOS. Alguns usuários relataram que estão enfrentando dificuldades para carregar o serviço, enquanto outros conseguem selecionar músicas, mas não conseguem reproduzi-las. Em alguns casos, as músicas podem ser reproduzidas no iOS, mas apenas se elas estiverem armazenadas em cache antes do problema surgir.

A empresa responsável pelo Spotify está trabalhando para resolver o problema, mas até o momento, não há uma previsão para quando o serviço regular será restaurado. O Spotify emitiu um comunicado afirmando que está analisando o caso e trabalhando para resolver o problema o mais rápido possível. Os usuários devem aguardar mais informações sobre a resolução do problema.

O problema afetou usuários em todo o mundo, com relatos de interrupção em diferentes países. A falta de acesso ao serviço de streaming de música causou transtorno para muitos usuários que dependem do Spotify para ouvir música. A empresa está trabalhando para resolver o problema e restaurar o serviço o mais rápido possível.
